# credit-risk-classification
Repo for Credit Risk Classification (Mod 20) Homework

Project Title: Credit Risk Classification
Project Description: In this Challenge, you’ll use various techniques to train and evaluate a model based on loan risk. You’ll use a dataset of historical lending activity from a peer-to-peer lending services company to build a model that can identify the creditworthiness of borrowers.

Using Code: Use Jupyter Notebook to analyze CSV data for credit risk classification information.

How to Use Project: Use Jupyter Notebook file to analyze CSV data.

Website used for troubleshooting code: https://github.com/JLeigh101/Credit-Risk-Classification/blob/main/credit_risk_classification.ipynb
